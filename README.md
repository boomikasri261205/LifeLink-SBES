# LifeLink SBES

LifeLink SBES is a Blood Emergency Support System developed to help people quickly find blood donors and hospitals during emergencies.

## Features

- Blood donor registration
- Blood request system
- Hospital database
- District-wise hospital search
- Firebase Firestore integration
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Firestore
